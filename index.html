HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title element appeared in the browser's title bar.
  • The section element is used to organize the content of html page into different categories.
  • The ul (which is know as unordered list) is used to list items without numbering on web page.
  • There are six different headings tags that are used to differentiate headings from normal text and decrease in size from h1 - h6.
  • The br (which is know as break element) element creates a line of empty space, or a line break, between two blocks of content.
  • The img (which is know as image element) element contains information about images that are displayed on the webpage.
  • The a (which is know as anchor element) creates links to the same webpage or other webpages.
  • The header element appears at the top of the page and usually contains a navigation menu and/or introductory content.
  • The src attribute defines the location, or source, of the image file.
  • The span tag is a non-semantic element used to create an inline division.
  • The div tag is a non-semantic element used to create page division.
  • The footer element appears at the bottom of the page and usually contains the author, copyright, contact, sitemap, and navigation.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • An attribute gives more details about an element.
  • In line styling is the best as it makes code reusable and easy to debug.
  • An '*' know as an asterics is used to assign a general attribute for the whole web page.
  • An 'id' attribute is used to label a specific element that need to take a personal styling.
  • A class attribute is used to assign group of elements that need to take the same attribute.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds all the changes to be staged for commit
  • git commit -m "commit message here": create a message to be seen on the gitHub as the commit message
  • git pull origin main: help to be sure the the local code is sync with the remote code.
  • git push origin (origin name): pushes the code in our local machine to the remote code on gitHub
  • git clone (url): help to copy the project from the remote location to our local machine.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An array is a single variable that is used to hold a group of data.
  • A function is different from a conditional statement because it has be called before it can work.
  • A function is a simply a set of instructions that tells the computer how to perform a certain task.
  • A for loop is method used to iterate over a series of values.
  • A JavaScript file can be identify by the extension(.js).
  • A variable is a container that allows us to store data in our code.
  • A variable can be decleared using 'var'.
  • if is a conditional statement that can be used to make decision in our code.
  • console.log() is used to output our code on the console and also use for debugging our code.
